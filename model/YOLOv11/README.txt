# YOLOv11 Pose Detection for Push-Ups

## Giới thiệu

Ứng dụng này sử dụng YOLOv11 để phát hiện tư thế trong động tác push-up. Mô hình được xây dựng dựa trên thư viện Ultralytics, cho phép theo dõi và đánh gía các tư thế thể dục (push up, squad, ...) dựa vào tính toán keypoints.

## Yêu cầu

- Python phiên bản từ 3.7 đến 3.11
- Thư viện Ultralytics

## Cài đặt

Để sử dụng ứng dụng, bạn cần cài đặt thư viện Ultralytics. Sử dụng lệnh sau để cài đặt:

```bash
pip install ultralytics
